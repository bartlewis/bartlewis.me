Types of support tools

Given the variety of support tools available, you can choose those that best meet your individual learning needs. Some are available online by clicking help & training at the top of each screen. Others are available in printed form.

Click each arrow below to learn more.

  • Online help
  • Web-based Training (WBT)
  • Getting Started Guide
  • User Guide
  • Policy links
Online help - Online help gives you information about MiSACWIS features and how to use them. It also contains detailed instructions for all of the important job tasks you may need to perform.

Online help is designed so that you can use it while you work. The help window is displayed on top of the MiSACWIS window so you can see it as you're working, and you can move the help window around for convenience.

Online help also includes a glossary—a searchable, alphabetical list of key MiSACWIS terms and their definitions. In addition, it includes brief Show Me animations.

Web-based training (WBT) - MiSACWIS provides access to interactive WBT.

The early lessons are aimed at orienting you to MiSACWIS. Later lessons are intended to reinforce classroom training and provide ongoing refresher training in common system tasks.

The WBT is very useful if you are just learning to use MiSACWIS or if you need to brush up on a particular area of the system.

Back to Bullets
Getting Started Guide - This is a document that provides basic information to help you begin using the system. It addresses common tasks that all users need to know to begin working in MiSACWIS.

The guide points you in the right direction when you need immediate assistance while becoming familiar with MiSACWIS.

Back to Bullets
User Guide - This document provides more detailed reference information describing aspects of system functions and operations in MiSACWIS.

It also describes the processes and step-by-step procedures you'll use to perform key tasks in MiSACWIS.

Back to Bullets
Policy links - MiSACWIS gives you access to the key policies that govern child welfare and juvenile justice work in the State of Michigan (SOM).

The only online help topics containing policy links are screen-level topics.

To view policy information, click a link in a screen-level topic. MiSACWIS displays a specific policy that applies to the screen you're working in.
